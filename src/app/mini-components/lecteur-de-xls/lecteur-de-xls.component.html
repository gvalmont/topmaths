Aucune donnée ne quitte votre ordinateur

Les fichiers sont lus et traités localement.
Aucune donnée ne transite par internet.
Les données sont toutes effacées dès que vous fermez l'onglet.

Comment le vérifier ?

Vous pouvez constater que les informations s'affichent instantannément.
Vous pouvez couper votre connexion à internet avant d'envoyer votre fichier et voir que ça ne change rien.

Pour les bidouilleurs

  • - Ouvrir la console avec F12
  • - Aller dans l'onglet Réseau / Network
  • - Vider son contenu (poubelle sur Firefox, panneau "sens interdit" sur Chrome)
  • - Envoyer un fichier, le manipuler
  • - Constater l'absence totale de mouvements sur le réseau
  • - Naviguer sur d'autres pages pour voir que l'onglet réseau fonctionnait bien correctement

Vous pouvez aussi consulter le code source du site.
